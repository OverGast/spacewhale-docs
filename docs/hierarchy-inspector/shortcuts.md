# Keyboard Shortcuts

## Style clipboard

These are scoped to the Hierarchy window and only fire when it has keyboard focus.

| Action | Windows / Linux | macOS |
| --- | --- | --- |
| Copy a row's style | ++ctrl+shift+c++ | ++cmd+shift+c++ |
| Paste style onto the selected rows | ++ctrl+shift+v++ | ++cmd+shift+v++ |

See [Copy & Paste Style](features/style-clipboard.md) for what the clipboard captures and what it deliberately skips.

## Theme assets

Unity's standard duplicate command, used in the Project window on a theme asset.

| Action | Windows / Linux | macOS |
| --- | --- | --- |
| Duplicate the selected theme asset | ++ctrl+d++ | ++cmd+d++ |

See [Themes & Preferences](features/themes.md) for how themes are created and switched.
